# FileShare 用户操作手册

## 一、快速开始

### 1.1 登录系统

1. 打开浏览器访问系统地址（如：`http://your-server:3000`）
2. 输入用户名和密码
3. 点击"登录"按钮

**默认管理员账号：**
- 用户名：`admin`
- 密码：`admin123`
- ⚠️ 首次登录后请立即修改密码

### 1.2 界面介绍

登录成功后，您将看到以下界面：

- **顶部导航栏**：显示系统名称和用户信息
- **左侧菜单**：功能导航
  - 工作台：查看系统概览
  - 文件管理：管理所有文件
  - 空间管理：管理团队空间
  - 系统管理：仅管理员可见
- **主内容区**：显示当前功能页面

## 二、文件管理

### 2.1 上传文件

1. 进入"文件管理"页面
2. 点击"上传文件"按钮
3. 选择要上传的文件（支持多选）
4. 等待上传完成

**注意事项：**
- 单文件最大支持10GB
- 支持断点续传
- 上传大文件时请保持网络连接稳定

### 2.2 下载文件

1. 在文件列表中，点击文件行的"下载"按钮
2. 文件将自动下载到本地

**权限说明：**
- 需要"读取"权限才能下载
- 如果设置了"禁止下载"权限，只能在线查看

### 2.3 搜索文件

1. 在文件管理页面顶部搜索框输入关键词
2. 系统将自动搜索文件名
3. 支持按文件类型、上传时间、上传人筛选

**高级搜索：**
- 在搜索框输入文件名关键词
- 支持模糊匹配
- 结果按相关性排序

### 2.4 文件操作

#### 重命名
1. 右键点击文件或点击操作按钮
2. 选择"重命名"
3. 输入新文件名
4. 确认保存

#### 删除文件
1. 点击文件行的"删除"按钮
2. 确认删除
3. 文件将移至回收站

#### 移动/复制文件
1. 选择文件
2. 点击"移动"或"复制"按钮
3. 选择目标文件夹
4. 确认操作

## 三、空间管理

### 3.1 空间类型

系统支持四种空间类型：

- **团队公共空间**：所有成员可见
- **部门空间**：部门成员可见
- **个人空间**：仅自己可见
- **项目专属空间**：项目成员可见

### 3.2 创建空间

1. 进入"空间管理"页面
2. 点击"创建空间"按钮
3. 填写空间信息：
   - 空间名称
   - 空间类型
   - 描述（可选）
4. 点击"确定"创建

**权限说明：**
- 只有管理员可以创建团队空间和部门空间
- 所有用户都可以创建个人空间

### 3.3 创建文件夹

1. 进入空间详情页
2. 点击"新建文件夹"按钮
3. 输入文件夹名称
4. 选择父文件夹（可选，创建子文件夹）
5. 确认创建

### 3.4 文件夹权限设置

1. 右键点击文件夹
2. 选择"权限设置"
3. 添加用户或用户组
4. 设置权限类型：
   - 读取：查看和下载
   - 写入：上传和修改
   - 删除：删除文件
   - 评论：添加评论
   - 下载：允许下载（可关闭）

## 四、文件共享

### 4.1 创建外部分享链接

1. 选择要分享的文件或文件夹
2. 点击"分享"按钮
3. 配置分享选项：
   - **访问密码**：设置4-16位密码（可选）
   - **允许邮箱**：限制特定邮箱访问（可选）
   - **有效期**：设置7天/30天/自定义
4. 生成分享链接
5. 复制链接发送给他人

### 4.2 访问分享链接

1. 打开分享链接
2. 如设置了密码，输入密码
3. 如限制了邮箱，输入允许的邮箱地址
4. 访问文件

### 4.3 管理分享

1. 进入"我的分享"页面
2. 查看所有创建的分享链接
3. 可以查看访问记录
4. 可以撤销分享

## 五、文件版本

### 5.1 查看历史版本

1. 打开文件详情页
2. 点击"版本历史"标签
3. 查看所有历史版本
4. 每个版本显示：
   - 版本号
   - 修改人
   - 修改时间
   - 文件大小

### 5.2 恢复历史版本

1. 在版本历史中选择要恢复的版本
2. 点击"恢复"按钮
3. 确认恢复
4. 文件将恢复到该版本

**注意事项：**
- 恢复操作会创建新版本
- 历史版本默认保留30天
- 管理员可配置保留天数

## 六、文件评论

### 6.1 添加评论

1. 打开文件详情页
2. 滚动到评论区域
3. 输入评论内容
4. 可以@其他成员（输入@符号后选择成员）
5. 点击"发送"

### 6.2 回复评论

1. 在评论下方点击"回复"
2. 输入回复内容
3. 发送回复

### 6.3 删除评论

- 只能删除自己的评论
- 管理员可以删除所有评论

## 七、权限管理

### 7.1 角色说明

系统预设四种角色：

- **管理员**：拥有所有权限
  - 用户管理
  - 权限配置
  - 系统设置
  - 日志审计

- **编辑者**：可以上传、修改、删除有权限的文件
  - 上传文件
  - 修改文件
  - 删除文件
  - 创建文件夹

- **查看者**：只能查看和下载
  - 查看文件
  - 下载文件
  - 无法修改或删除

- **仅评论者**：只能查看和评论
  - 查看文件
  - 添加评论
  - 无法下载或修改

### 7.2 申请权限

1. 找到需要访问的文件或文件夹
2. 点击"申请权限"
3. 填写申请理由
4. 提交申请
5. 等待管理员审批

### 7.3 管理员权限设置

1. 进入"系统管理" > "用户管理"
2. 选择用户
3. 修改用户角色
4. 或进入"权限管理"设置文件夹权限

## 八、个人设置

### 8.1 修改密码

1. 点击右上角用户头像
2. 选择"个人设置"
3. 在"修改密码"区域：
   - 输入旧密码
   - 输入新密码（至少6位）
   - 确认新密码
4. 点击"修改密码"

### 8.2 查看个人信息

在个人设置页面可以查看：
- 用户名
- 邮箱
- 角色
- 注册时间

## 九、常见问题

### Q1: 上传文件失败？
**A:** 检查以下几点：
- 文件大小是否超过10GB限制
- 是否有上传权限
- 网络连接是否稳定
- 磁盘空间是否充足

### Q2: 无法下载文件？
**A:** 可能原因：
- 没有读取权限
- 设置了"禁止下载"权限（只能在线查看）
- 文件已被删除

### Q3: 找不到文件？
**A:** 尝试：
- 使用搜索功能
- 检查是否在回收站
- 检查是否有访问权限
- 联系管理员

### Q4: 忘记密码？
**A:** 联系系统管理员重置密码。

### Q5: 如何批量上传文件？
**A:** 在文件上传时，可以按住Ctrl（Windows）或Cmd（Mac）选择多个文件。

### Q6: 回收站文件如何恢复？
**A:** 
1. 进入回收站（管理员功能）
2. 选择要恢复的文件
3. 点击"恢复"按钮

### Q7: 分享链接过期了怎么办？
**A:** 联系文件所有者重新创建分享链接。

## 十、最佳实践

### 文件命名
- 使用有意义的文件名
- 遵循团队命名规范
- 避免使用特殊字符

### 文件组织
- 按项目或部门创建空间
- 使用文件夹分类管理
- 定期清理无用文件

### 权限管理
- 最小权限原则
- 敏感文件单独设置权限
- 定期审查权限设置

### 数据安全
- 重要文件及时备份
- 使用强密码
- 定期修改密码
- 不在外部分享敏感文件
